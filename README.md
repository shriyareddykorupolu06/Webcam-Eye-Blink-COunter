# Webcam-Eye-Blink-Counter
## Overview
This project detects and counts the eye-blinks in real time using a webcam
## Features
Real-time webcam face tracking
Eye blink detection
Blink counter displayed on screen
## Technologies Used
HTML
JavaScript
MediaPipe FaceMesh
## How to RUn
Open project in VS Code
Right-click on index.html-Open with Live Server
Allow webcam permission
Blink to see counter increase
## Notes 
Runs completely in browser
No dataset required
Implementation and testing done by me with AI assistance.
